During the first week of class we will begin exploring ideas for your **term project**. You do not need to have a final project topic yet, but you should start thinking about technologies or questions that interest you.

This activity will help you:

- explore the frontend ecosystem
- practice short technical explanations

---

# Step 1 — Lightning Talk Demonstration

A lightning talk is a **5–8 minute explanation** of a technology or concept. A good lightning talk answers four basic questions:

1. **What is it?**
2. **Why does it exist?**
3. **How does it work?**
4. **What are the trade-offs?**

Example topic:

```
Vite
```

Example structure:

**What problem does it solve?**  
Traditional build tools can be slow during development.

**How does it work?**  
Vite uses native ES modules in the browser instead of bundling everything first.

**Advantages**  
Fast development startup and hot module replacement.

**Trade-offs**  
Different plugin ecosystem compared to Webpack.

# Step 4 — Generating Project Ideas

Now we will begin thinking about **term project ideas**.

A good project usually investigates a **specific technical question**.

Example:

```
Technology
Vite

Question
Is Vite significantly faster than Webpack during development?

Experiment
Measure development server startup time.
```

Your task:

Write down **three possible project questions** related to frontend technology.

Good projects often involve:

- comparing technologies
- measuring performance
- experimenting with frameworks
- exploring architectural trade-offs
